# paso 0 (si no lo tenemos)
1,- Descargar, instalar git
2.- Descargar, instalar VSC(Visual Estudio Code)
3.- Crear un directorio y abrir VSC
4.- Abrir la terminal y probar comandos:
    git -v ==> para ver la version del git
    git config --global user.name "nombre" ==> agregar nombre de usuario
    git config --global user.email "correo" ==> agrega correo del usuario
5.- Clonar el repositorio creado en GitHub:
    git clone: url http://.....git ==> url del repositorio
# Paso 1
Cada cambio que hacemos dentro del proyecto local cuando queremos guardar
debemos hacer lo siguiente:
    git add . ==> prepara los archivos
    git commit -m "Comentario del avance"==> crea un punto de guardado
    git push origin main ==> sube los cambios al respositorio den la nube de GH
